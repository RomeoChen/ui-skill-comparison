什么是 Composition API

Composition API 是 Vue 3 最核心的特性之一，它提供了一种全新的组件逻辑组织方式... 

import { ref, computed, watch } from 'vue'

export default {
  setup() {
    const count = ref(0)
    const doubled = computed(() => count.value * 2)
    
    watch(count, (newVal) => {
      console.log('count changed:', newVal)
    })
    
    return { count, doubled }
  }
}
"Composition API 不仅是一种语法，更是一种思维方式。"
