什么是 Composition API

Composition API 是 Vue 3 引入的一种新的组件逻辑组织方式... 

import { ref, computed } from 'vue'

export default {
  setup() {
    const count = ref(0)
    const doubled = computed(() => count.value * 2)
    
    return { count, doubled }
  }
}
Composition API 让逻辑复用变得前所未有的简单。
